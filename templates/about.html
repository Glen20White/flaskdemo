About This App

This website allows you to search for Wikipedia pages and view key information, such as the page title, summary, and a link to the full article.

It uses the Wikipedia API (via the wikipedia Python module) and is built with the Flask web framework.

Features

  • Search by page title or keywords
  • View summary text and direct page link
  • Handles disambiguation automatically
  • Keeps your last search in session storage

This project was created for learning purposes and demonstrates basic Flask routing, templates, and API usage.
